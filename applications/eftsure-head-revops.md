### Q3: Forecasting approach

I use a Commit-Expected-Best model:

- **Commit**: 100% confident will close (verbal/signed)
- **Expected**: Weighted by stage, rep history, deal health
- **Best**: Upside scenario

Built this in HubSpot with custom fields. Every week, we knew exactly what we'd close - not a range.

At Penbrothers, this was a key driver of the 3x growth. Leadership trusted the numbers. Investors trusted the numbers.

Also use advanced Excel/Sheets for scenario modeling - capacity planning, territory analysis, quota allocation.

## Automation Ideas

### 1. Multi-Region Lead Routing

HubSpot workflow that auto-routes leads by geography + sets local follow-up times.

**Steps:**
1. Create "region" property (AU/US/EU)
2. Build workflow: if domain ends .au → region = AU, assign to AU rep
3. Create 3 separate email sequences with local send times
4. Add task creation for each new lead

**Time**: 2-4 hours
**ROI**: 20-30% faster first response

### 2. CRM Hygiene Suite

Weekly automated checks for duplicates, incomplete records, stale data.

**Steps:**
1. Enable built-in duplicate detection in HubSpot
2. Create "data completeness score" property (0-100 based on filled fields)
3. Build weekly workflow: if score < 50, create task for owner
4. Set up "stale deal" workflow: if no activity in 30 days, flag it

**Time**: 3-5 hours
**ROI**: Cleaner data = reliable forecasts

### 3. Cross-Region Forecasting Dashboard

Unified view of pipeline + automated alerts.

**Steps:**
1. Create custom properties: forecast_category (Commit/Expected/Best), region_pipeline
2. Build dashboard with: total pipeline, by region, by forecast category
3. Add deal velocity metrics (avg time to close by region)
4. Set up alerts: if Commit < 80% of quota → email Sales Leader

**Time**: 4-6 hours
**ROI**: Real-time visibility, no more manual spreadsheets
