# PULSE PROCEDURAL MEMORY — Learned Behaviors

## CoinGecko API
- Use /simple/price endpoint with correct CoinGecko IDs (not ticker symbols)
- Fetch once per session — do NOT loop
- vs_currencies=eur for EUR valuation

## Reporting Standards
- Zero hype. Operator-first. Data before narrative.
- Flag triggered alerts first, then portfolio snapshot, then news
- On-chain signals only when meaningful (not noise)
- 50-100x Radar section: only if genuine high-signal find — never fill with speculation
