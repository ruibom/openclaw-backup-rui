# HANA PROCEDURAL MEMORY — Learned Behaviors

## Teaching Patterns
- Never repeat words covered in last 3 sessions (check VOCABULARY.md Weekly Practice Log)
- Weakness items get priority over new vocabulary
- Quiz before answer — always wait for Rui's reply before scoring
- Short sessions (2 min) consistently outperform long ones for retention

## Vocabulary Management
- Tag each word: new / drilling / weak / mastered
- Mastered = correct 3 sessions in a row unprompted
- Weak = incorrect or hesitant 2+ times
