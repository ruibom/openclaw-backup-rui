# SKILL: Web Search and Fetch

## When to use
Any time you need to research a person, company, or URL. This is your PRIMARY research tool for all inbound leads. Use it without hesitation — it is always available.

## web_search
Use for: finding people, companies, news, funding rounds, acquisitions, LinkedIn profiles.
- Run multiple targeted searches — one search is rarely enough
- Search the person AND the company separately
- Example queries: "Dimitris Maniatis CEO", "Buildcollective.io", "Upstream Systems acquisition 2025"

## web_fetch
Use for: retrieving the full content of a specific URL.
- Use after web_search to read full articles, job postings, or company pages

## Critical rule
NEVER use the Chrome browser tool for research. NEVER stop because the browser is unavailable.
web_search and web_fetch are always available. Use them for all research tasks.

## Output
Summarize what you found. Cite the source. Never present assumptions as facts.
