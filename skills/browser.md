# SKILL: Browser Tool

## When to use
Any time you receive a URL. Never guess or assume page content.

## How to use
1. Call the browser tool with the exact URL provided.
2. Wait for full page load before reading content.
3. If page fails to load → report exact error to Rui, do not proceed.
4. If page shows "job closed", "removed", "expired", "404" → report immediately, do not proceed.
5. Never summarize a page you haven't actually fetched.

## Output
Return the real page title and confirm load success before doing anything else.
