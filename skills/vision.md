# SKILL: Vision Tool (Images & Screenshots)

## When to use
Any time Rui pastes or uploads an image, screenshot, or photo.

## How to use
1. Call the vision tool on the image immediately.
2. Extract all readable text verbatim.
3. Describe visual content: layout, charts, UI elements, diagrams.
4. Identify key points and data.
5. Never describe an image you haven't actually analyzed.

## Output
Return extracted text + visual description before summarizing.
